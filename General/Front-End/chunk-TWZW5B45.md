# chunk-TWZW5B45.js

## Overview

A foundational **JavaScript utility chunk** generated by the bundler. It provides low-level object manipulation helpers and a lightweight CommonJS-compatible module wrapper used by other chunks.

## What It Contains

### Object Utility Functions

These mirror the standard patterns used by transpiled TypeScript/ESNext code:

| Variable | Native Equivalent | Purpose |
|----------|------------------|---------|
| `g` | `Object.defineProperty` | Define a single property on an object |
| `r` | `Object.defineProperties` | Define multiple properties at once |
| `s` | `Object.getOwnPropertyDescriptor` | Get a property's descriptor |
| `t` | `Object.getOwnPropertyDescriptors` | Get all property descriptors |
| `u` | `Object.getOwnPropertyNames` | List own (non-symbol) property names |
| `h` | `Object.getOwnPropertySymbols` | List own symbol-keyed properties |
| `v` | `Object.getPrototypeOf` | Get the prototype of an object |
| `k` | `Object.prototype.hasOwnProperty` | Check for own property existence |
| `m` | `Object.prototype.propertyIsEnumerable` | Check if a property is enumerable |

### Spread / Assign Helpers

| Variable | Purpose |
|----------|---------|
| `l` | Define or assign a single property (handles both `in` check and direct assignment) |
| `w` | Shallow object spread — copies own enumerable string and symbol keys |
| `x` | Spread using `defineProperties` (preserves descriptors) |
| `z` | Object rest — copies all keys **except** a given exclusion list |

### Module System (`y`)

A minimal `require`-like wrapper that delegates to the real `require` if available, or throws a descriptive error in ESM-only environments. Used to support legacy CommonJS modules included in the bundle.

## Role in the Bundle

`chunk-TWZW5B45.js` is imported by every other chunk. It is the lowest-level dependency in the dependency graph and must load first.
