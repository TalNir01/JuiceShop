# chunk-T3PSKZ45.js

## Overview

A **shared utility chunk** that provides the same low-level JavaScript object manipulation helpers as `chunk-TWZW5B45.js`. It is a second utility layer generated by the bundler to split large dependency graphs, reducing individual chunk sizes and improving parallel loading in the browser.

## Contents

### Object Utility Functions

Mirrors the utilities in `chunk-TWZW5B45.js`:

| Variable | Native Equivalent | Purpose |
|----------|------------------|---------|
| `g` | `Object.defineProperty` | Define a property with a full descriptor |
| `r` | `Object.defineProperties` | Define multiple properties at once |
| `s` | `Object.getOwnPropertyDescriptor` | Retrieve a property descriptor |
| `t` | `Object.getOwnPropertyDescriptors` | Retrieve all property descriptors |
| `u` | `Object.getOwnPropertyNames` | List own string-keyed property names |
| `h` | `Object.getOwnPropertySymbols` | List own symbol-keyed properties |
| `v` | `Object.getPrototypeOf` | Get the prototype chain parent |
| `k` | `Object.prototype.hasOwnProperty` | Own-property existence check |
| `m` | `Object.prototype.propertyIsEnumerable` | Enumerability check |

### Spread / Assign Helpers

| Variable | Purpose |
|----------|---------|
| `l` | Assign or define a single property |
| `w` | Shallow spread — copies own enumerable string + symbol keys |
| `x` | Descriptor-preserving spread via `defineProperties` |
| `z` | Object rest — copies keys excluding a provided list |

### Module Wrapper (`y`)

A `require`-compatible shim identical to the one in `chunk-TWZW5B45.js`, supporting CommonJS modules embedded in the ESM bundle.

## Why Two Utility Chunks?

The bundler (esbuild) splits helpers into separate chunks when different parts of the dependency graph need them independently. Having two utility chunks prevents circular dependencies and allows the browser to cache them separately — if `chunk-TWZW5B45.js` changes, `chunk-T3PSKZ45.js` (and the modules that depend on it) remain cached.

## Consumers

Imported by `chunk-T3PSKZ45.js`'s dependents via `main.js`:

```js
import { a as Am } from "./chunk-T3PSKZ45.js";
```
