# main.js

## Overview

The primary Angular application entry point for OWASP Juice Shop's front-end. This is a bundled, minified output file generated by the Angular build toolchain (esbuild).

## Role in the Application

`main.js` is the top-level bundle that bootstraps and wires together the entire Juice Shop single-page application. It imports symbols from all other chunk files and combines them into the running Angular app.

## What It Contains

- **Application bootstrap logic** — initializes the Angular platform and root module
- **Component definitions** — Angular components for pages such as the product list, login, registration, basket, administration panel, and more
- **Service implementations** — HTTP services for interacting with the Juice Shop REST API (`/api/`, `/rest/`)
- **Routing configuration** — maps URL paths to components
- **Helper utilities** — polyfill shims for `typeof`, class definition helpers (`HS`, `NS`, `RS`), and object spread (`Ot`)

## Notes

- This file is minified and tree-shaken; symbol names (e.g. `Am`, `bm`) are mangled by the bundler.
- Directly editing this file is not recommended — changes should be made to the Angular source and rebuilt.
